Display a daily verse from the Quran on your WordPress site using Elementor, Gutenberg, Classic Editor, or a shortcode.

== Description ==
The **Daily Quran Verse** plugin allows you to display a daily verse from the Quran on your WordPress website. The verse updates automatically every day and can be integrated easily using Elementor, Gutenberg, Classic Editor, a shortcode, or a sidebar widget.

**Features:**

- **Daily Quran Verse**: Automatically updates with a new verse from the Quran each day.
- **Elementor Widget**: Easily add the daily verse to your Elementor pages.
- **Gutenberg Block**: Seamlessly integrate the daily verse into the Gutenberg editor.
- **Classic Editor Support**: Add the daily verse to posts and pages using the Classic Editor.
- **Shortcode**: Use `[daily_quran_verse]` to display the verse anywhere on your site.
- **Sidebar Widget**: Show the daily verse in your site's sidebar.
- **Social Sharing**: Share the verse on Facebook, Twitter, and WhatsApp.
- **Copy Verse**: Easily copy the verse to your clipboard.
- **License Option**: Choose from different licensing options for customization and redistribution.

== Installation ==

1. **Upload the Plugin Files**: Upload the plugin files to the `/wp-content/plugins/daily-quran-verse` directory, or install the plugin through the WordPress plugins screen directly.
2. **Activate the Plugin**: Activate the plugin through the 'Plugins' screen in WordPress.
3. **Configure Settings**: Go to the plugin settings page (if available) to configure any options.

== Usage ==

### Elementor
1. Open the Elementor editor for the page where you want to add the daily verse.
2. Search for the **Daily Quran Verse** widget in the Elementor panel.
3. Drag and drop the widget to your desired location on the page.

### Gutenberg
1. Open the Gutenberg editor for the post or page where you want to add the daily verse.
2. Click on the **Add Block** button.
3. Search for the **Daily Quran Verse** block and add it to your content.

### Classic Editor
1. Open the Classic Editor for the post or page where you want to add the daily verse.
2. Use the shortcode `[daily_quran_verse]` to display the verse.

### Shortcode
Use the shortcode `[daily_quran_verse]` anywhere on your site to display the daily Quran verse.

### Sidebar Widget
1. Go to the **Widgets** section in your WordPress admin.
2. Add the **Daily Quran Verse** widget to your desired sidebar.
